# Style Process (공정 정보)

이 페이지는 특정 스타일에 대한 공정 목록을 관리합니다.

## 주요 기능

-   **공정 목록 조회:** 특정 스타일에 등록된 공정들의 목록을 보여줍니다.
-   **공정 정보 표시:** 각 공정은 다음 정보를 포함합니다.
    -   **공정 이름:** 예) 주머니 달기, 지퍼 달기
    -   **표준 공정 시간 (SMV - Standard Minute Value):** 해당 공정을 완료하는 데 걸리는 표준 시간입니다.
    -   **산출 공정 시간 (ETD - Estimated Time of Departure):** 실제 작업 데이터를 기반으로 산출된 예상 완료 시간입니다. 초기에는 SMV와 같거나 비어있을 수 있으며, 추후 생산 데이터가 쌓이면 자동으로 계산되어 업데이트될 예정입니다.
-   **공정 추가/수정/삭제:** 새로운 공정을 추가하거나 기존 공정의 정보를 수정하고 삭제할 수 있는 기능이 제공될 예정입니다.

## 데이터 모델 예시

| 필드              | 설명                       | 예시        |
| ----------------- | -------------------------- | ----------- |
| `id`              | 공정 고유 ID               | `P-001`     |
| `name`            | 공정 이름                  | "주머니 달기" |
| `smv`             | 표준 공정 시간 (초)        | 10          |
| `etd`             | 산출 공정 시간 (초)        | 12          |

## 향후 개발 계획

-   생산 관리 시스템과의 연동을 통해 실제 작업 시간을 수집하고, 이를 바탕으로 산출 공정 시간(ETD)을 자동으로 계산하고 업데이트하는 기능을 구현할 예정입니다.
